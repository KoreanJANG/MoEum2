  1. 데이터 저장
    • 모은 링크는 안전하게 저장되나요?

      : 모음은 강력한 개인정보보호정책을 기반으로 사용자의 모든 데이터를 안전하게 보호하고 있습니다.

    • 데이터 저장 기간이 별도로 정해져있나요?

      : 저장 데이터(링크, 메모, 질문 글, 댓글 등)는 사용자가 직접 영구 삭제하기 전까지 암호화된 장소에 보관됩니다. 사용자가 모음 서비스를 탈퇴하는 경우에도 등록된 저장 데이터는 자동으로 삭제되지 않으며, 회원 정보 삭제로 인한 컨텐츠 편집 및 삭제 요청 처리는 불가능합니다. 삭제를 희망하는 경우, 본인의 저장데이터를 직접 삭제 하신 후 탈퇴를 진행해주시길 바랍니다. ※저장데이터와 별개로 개인정보는 개인정보처리방침에 따라 탈퇴 즉시 삭제됩니다.

  1. 계정관리 및 탈퇴
    • 탈퇴는 어떻게 하나요?

      : 설정> 계정관리> 회원탈퇴를 선택하여 탈퇴할 수 있습니다. 탈퇴 시 모음 서비스 이용에 제한이 생길 수 있습니다.

🖇️나의 저장

  1. 저장 방법
    • 모음 앱에 저장은 어떻게 하나요?

      1) + 버튼 >복사한 링크 붙여넣기> 메모, 테마 입력(선택사항)> 등록 클릭

      2) 저장하고 싶은 앱, 웹 화면에서 추가 메뉴 버튼> 공유 버튼 클릭> 모음 앱 클릭> 저장완료

  1. 저장 오류
    • 저장방법 1을 통해 저장하였는데 나의 저장 목록에 뜨지 않아요! 왜 그럴까요?

      : 해당 링크 주소가 유효하지 않거나 저장 시점의 일시적인 서버 문제로 정상적으로 저장이 이루어지지 않았을 수 있어요. 더 자세한 진단을 위해서 ( 여기 📍)로 오류 접수를 부탁드려요. 해당 접수를 통해 더 나은 서비스 제공을 위해 노력하겠습니다.

🗨️ 커뮤니티

  1. 트렌드 레포트
    • 트렌드 레포트가 무엇인가요?

      트렌드 레포트란 사용자들의 저장 데이터를 바탕으로 모음 유저들이 현재 관심있는 위시 상품, 관심사들을 다루는 섹션입니다.

    • 트렌드 레포트는 언제 발행되나요?

      현재는 2주에 1회 발행되고 있습니다. 주제는 내부 기준을 거쳐 선정되고 있습니다. 레포트에서 보고 싶으신 주제가 있다면 언제든지 moum.help@gmail.com으로 의견 주세요😃

  1. 질문하기
    • 질문하기 게시글은 누구나 볼 수 있나요?

      네. 질문하기 게시글은 전체 공개 영역으로 모음 유저 전체가 둘러볼 수 있는 영역입니다.

    • 질문하기 게시글 작성 시 링크는 어떻게 첨부하나요?

      : 나의 위시 > 해당 링크의 오른쪽 … ※ 첨부할 수 있는 링크의 최대 개수는 1개이며 추후 링크 갯수는 확대 될 예정입니다.

  1. 검색 방법
    • 검색 결과가 없는 경우가 있나요?

      : 네. 한번도 해당 검색어가 포함된 링크가 저장이 되지 않았거나, 질문하기 게시글이 없는 경우에는 검색 결과가 나오지 않습니다.

    • 검색 결과를 저장할 수 있나요?

      : 네, 가능합니다. 검색결과에 표시된 트렌드, 질문하기 내 링크 공유> 나의 저장 클릭
